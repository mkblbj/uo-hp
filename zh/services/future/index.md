株式会社UO 将基于在乐天市场和 Amazon 积累的 **EC 运营经验**，持续扩大经营品类，并加强包含中日协作在内的业务联动。  
今后还将作为现有业务的延伸，推进 **利用 AI 与 IT 支援 EC 运营** 以及 **业务支援 Agent 的开发**，进一步强化业务基础。

## 商品领域的扩展

今后除现有手机配件外，公司还计划拓展到以下 **有助于日常生活的商品**。

- 食品与酒类
- 电器产品
- 文具
- 服装

## 支撑 EC 运营的 AI 与 IT 应用

基于既有 EC 业务中积累的运营经验，今后公司也将推进 **利用 AI 与 IT 的业务支援**。  
具体而言，计划从更贴近 EC 一线的领域开始务实应用，例如商品信息整理、页面优化、客户服务辅助以及运营效率提升等。

## 业务支援 Agent 的开发

从中长期来看，公司也将把 **支援 EC 日常运营的 IT 工具与 Agent 开发** 纳入视野。  
在活用内部积累的销售运营经验的同时，推动建设能够支撑商品理解、咨询应对与业务整理的机制，并将其培育为 **支撑 EC 业务的实务基础**。

## 既有经验的活用

- 来自乐天市场与 Amazon 的 **销售运营经验**
- 商品页优化与评价应对经验
- 新品类启动时的运营设计
- 基于销售一线经验形成的 **业务改善视角**

## 强化中日协同

株式会社UO 不仅着眼于日本国内销售，也在推进包含 **与中国合作方之间的接单、发单及商品上架协作** 在内的业务扩张。  
今后也将配合商品领域扩展，持续强化 **包含中日协同在内的业务基础**。
