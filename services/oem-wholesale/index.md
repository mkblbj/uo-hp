# OEM・卸 / 越境連携

株式会社UOは、自社EC運営で蓄積した知見をもとに、法人向けのOEM・加工・卸・越境連携にも対応しています。販売現場を持つ会社として、商品企画から納品までを現実的な運用目線で設計できることが強みです。

## 対応領域

- スマートフォン全機種ケースのOEM・加工・卸・販売
- 中国からのオーダーメイド直輸入卸・販売
- 日本から中国への出品支援
- 小ロットでの試験導入から継続運用までの対応

## 対応体制

- 最低ロットは主要商材で1枚から対応可能
- 対応言語は日本語 / 中国語 / 英語
- 物流は日本郵便 / 佐川急便を利用
- 不良品対応は再送または値引きを基本方針とする
- 支払い方法は銀行振込 / クレジットカードに対応

## 株式会社UOが提供できること

- 販売現場の声を踏まえた商品仕様のすり合わせ
- 多機種対応や新機種対応を見据えた商品展開
- 加工や検品を含む実務フローの整理
- 日本市場と中国側の連携をまたぐ運営支援
