# 販売実績

<PerformanceAwardBadges />

2019年に楽天市場へ出店して以来、株式会社UOの販売店は多くのお客様に利用されてきました。  
ここでは、販売実績と評価指標を通じて、スマートフォンアクセサリー事業における運営基盤の強みを整理しています。

::: info このページでわかること
- 楽天市場を中心とした主な販売実績
- 評価指標から見える顧客信頼
- 実績を支える運営面の強み
:::

## 主な実績

| 項目 | 内容 |
| --- | --- |
| 出店開始 | 2019年に楽天市場へ出店 |
| 2021年売上（実績） | 1億円 |
| 口コミ総合評価 | 4.59（2026年3月31日時点） |
| 口コミ件数 | 16,755件（2026年3月31日時点） |
| 楽天市場内ランキング（過去実績） | 全ショップ中31位獲得 |

> ※ 口コミ総合評価と口コミ件数は、松武商店（楽天市場）の公開情報をもとにした2026年3月31日時点の数値です。  
> ※ 出店開始、2021年売上、楽天市場内ランキングは、これまでの実績を示しています。

## 実績から見えること

- 出店開始後、比較的早い段階で**売上規模の拡大**を実現しています
- **評価点数**と**口コミ件数**の両面から、継続的な顧客満足の積み重ねがうかがえます
- 単なる販売量だけでなく、**店舗運営**と**顧客対応**を含めた総合力が実績につながっています

## 実績を支える運営力

株式会社UOでは、商品を販売するだけでなく、商品ページの改善、レビューの蓄積、顧客対応の質の向上を継続して行ってきました。  
こうした日々の運営改善が、販売実績と評価の両面につながっています。

特に、スマートフォンアクセサリー事業では、**多機種対応**、**小ロット対応**、**加工から販売までをつなぐ体制**を整えながら、継続的な店舗運営を行ってきたことが基盤となっています。
