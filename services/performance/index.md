# 販売実績

2019年に楽天市場へ出店して以来、株式会社UOの販売店は多くのお客様に利用されてきました。ここでは、会社案内資料に記載されている主な実績を整理しています。

## 主な指標

| 指標 | 内容 |
| --- | --- |
| 出店開始 | 2019年に楽天市場へ出店 |
| 2021年売上 | 1億円 |
| 口コミ総合評価 | 4.67 |
| 口コミ件数 | 12,546件 |
| 楽天市場内ランキング | 全ショップ中31位獲得 |

## 実績から見えること

- 立ち上げから比較的短期間で売上規模を拡大している
- 評価点数とレビュー件数の両面で、顧客満足度の高さがうかがえる
- 販売だけでなく、運営改善や顧客対応の積み重ねが実績につながっている
