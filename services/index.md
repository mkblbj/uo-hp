# 事業案内

株式会社UOは、スマートフォンアクセサリー事業で培ったEC運営力を基盤に、国内産食品のネット販売、OEM・卸、越境連携へと事業領域を広げています。このセクションでは、各事業の役割と、実際の取扱商品・実績・今後の展開をまとめています。

## 事業の柱

- [スマートフォンアクセサリー事業](./mobile-accessories/)：松武商店を中心に、多機種対応・オーダーメイド・小ロット対応を強みに展開しています。
- [国内産食品事業](./domestic-foods/)：国内産食品の価値を再発見し、消費者へ届ける新たな販売事業です。
- [OEM・卸 / 越境連携](./oem-wholesale/)：加工・卸・直輸入・海外向け出品まで含めた法人向けの対応領域です。

## 商品・実績・展開

- [主要商品](./products/)：スマホケースの主力商材と対応条件を整理しています。
- [販売実績](./performance/)：楽天市場での販売実績や評価指標を掲載しています。
- [選ばれる理由](./strengths/)：多機種対応、品質、顧客対応という強みをまとめています。
- [今後の展開](./future/)：食品、雑貨、越境連携を含む今後の方向性を整理しています。
