# 事業案内

株式会社UOは、**スマートフォンアクセサリー事業で培ったEC運営力**を基盤に、**OEM・加工・卸販売**、**国内産食品のEC展開**、**越境連携**へと事業領域を広げています。  
本ページでは、各事業の役割と特徴、関連する商品・実績・今後の展開をまとめています。

:::

## 主要事業

- [スマートフォンアクセサリー事業](./mobile-accessories/)  
  松武商店を中心に、多機種対応・オーダーメイド・小ロット対応を強みに展開している中核事業です。

- [国内産食品事業](./domestic-foods/)  
  日本各地の農産物や加工品の価値を見直し、消費者へ届ける新たな販売事業です。

- [OEM・卸 / 越境連携](./oem-wholesale/)  
  加工・卸・直輸入・海外向け出品までを含めた、法人向け対応領域を整理しています。

## 販売チャネルと展開基盤

株式会社UOは、複数のECチャネルを通じて販売基盤を築いてきました。  
スマートフォンアクセサリー事業を中心に、国内販売と越境連携の両面から事業を展開しています。

- **楽天市場**：松武商店、3911、十色生活、天海スポーツ、HAGUMI、松田、ALLCase
- **Amazon**：UOWORLD3911、幸田良品、国内産屋
- **メルカリ / Qoo10 / au PAY マーケット**：いずれも3911で展開
- **越境連携**：中国向け出品や中国側の生産連携にも対応
